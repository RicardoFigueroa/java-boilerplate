**java-boilerplate**

**Build**

`mvn clean install`

**Run**

`java -jar java-boilerplate-1.0-SNAPSHOT.jar`

**Test**

- http://localhost:8080/greeting
- http://localhost:8080/greeting?name=User
- http://localhost:8080/getPerson?id=1
- http://localhost:8080/getPerson?id=2

- Java 8
- Spring Boot
- Hibernate / JPA
